# Frontend (EventIQ)

React + Vite + Tailwind UI for EventIQ.

## Prerequisites

- Node.js 20+
- npm 10+

## Setup

1. Copy `.env.sample` to `.env`
2. Install dependencies:
   - `npm install`
3. Run dev server:
   - `npm run dev`

## Environment Variables

- `VITE_API_BASE_URL`: API Gateway base URL (default local: `http://localhost:4000`)

## Build

- `npm run build`
- `npm run preview`
